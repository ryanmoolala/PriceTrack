# PriceTrack

Description:
PriceTrack is a web application designed to provide real-time data on stock prices. It ensures users have the most up-to-date information on their favorite stocks.

Tech Stack used: 
<br/>

Frontend / Client
- HTML
- Tailwind CSS
- JavaScript
- ReactJS

</br>

Backend / Server
- TypeScript
- NodeJS
